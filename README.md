# 📌 Project Overview

This is a School Budget Management System developed to simplify budget planning and procurement processes. The system allows users to manage school budgets, procurement requests, vendors, and reports through an easy-to-use interface.

## ✨ Features

* Secure user authentication and login system.
* Manage schools, budget heads, items, and vendors.
* Create and manage budget allocations.
* Generate procurement requisitions and quotations.
* Automatically calculate and select the lowest vendor quotation.
* Generate procurement-related PDF documents and reports.
* Simple and user-friendly dashboard for managing daily operations.

## 🛠️ Tech Stack

* **Backend:** PHP
* **Database:** MySQL
* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **PDF Libraries:** FPDF, FPDI, DomPDF

## 👨‍💻 My Role

I developed and maintained this project, implementing the database design, backend business logic in PHP, frontend interfaces using Bootstrap and JavaScript, and automated PDF document generation. I also worked on procurement workflows, budget management, vendor management, and system optimization.
